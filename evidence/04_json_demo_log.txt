Step 4: JSON Formatting Demo

[Valid JSON] Pass
Input : {"name": "Alice", "age": 30}
Output: {
  "name": "Alice",
  "age": 30
}

[Markdown fences] Fixed
Input : ```json
{"name": "Bob"}
```
Output: {
  "name": "Bob"
}

[Single quotes] Fixed
Input : {'name': 'Charlie', 'score': 95}
Output: {
  "name": "Charlie",
  "score": 95
}

[Trailing comma] Fixed
Input : {"key": "value",}
Output: {
  "key": "value"
}

[Truly invalid] Fixed
Input : This is not JSON at all: ??? {]
Output: {"error": "Invalid JSON after repair", "raw": "This is not JSON at all: ??? {]"}
